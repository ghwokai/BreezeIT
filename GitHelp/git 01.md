#git 01.md
##git config
1.user config
```
    $ git config --global user.name "name"
    $ git config --global user.email "email@xxx.com"
```
2.editor
```
    $ git config --global core.editor xxxeditor
```
3.config
```
    $ git config --list
```
git config \<key\>  

4.alias
```
    $ git config --system alias.st staus
    $ git config --system alias.ci commit
```
and so on  
  
5.git help
```
    $ git help <verb>
    $ git <verb> --help
    $ man git-<verb>
    
    $git help config
```
