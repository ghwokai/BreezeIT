##git staging area
1.git log
```
    $ git log -stat
    $ git log --pretty=oneline
```
2.git diff
```
    $ git diff
    $ git diff HEAD
```
3.git status
```
    $ git status -s
```
4.git clone [URL]
```
    $ git clone https://github.com/libgit2/libgit2

    $ git clone https://github.com/libgit2/libgit2 mylibgit
```
